## Gradle コマンド

1. 実行（正しくは、ビルドして実行してる）

```
./gradlew bootRun
```

2. ビルド

```
./gradlew build
```

3. 依存関係を出力

```
./gradlew dependencies
```

4. 利用可能なタスクを出力

```
./gradlew tasks
```

5. タスクの実行

```
./gradlew <task-name>
```

6. バージョン周りを出力

```
./gradlew --version
```
